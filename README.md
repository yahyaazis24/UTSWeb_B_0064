# FitLife Gym - Company Profile Website

## Deskripsi Project

FitLife Gym merupakan website company profile yang dibuat untuk memperkenalkan pusat kebugaran modern dengan fasilitas lengkap, trainer profesional, serta berbagai pilihan program latihan dan membership.

Website ini dibuat menggunakan HTML, CSS, Bootstrap, dan JavaScript dengan tampilan yang responsif serta fitur interaktif berbasis manipulasi DOM.
